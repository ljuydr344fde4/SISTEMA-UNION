PROCEDIMENTO PARA
SNMP VOLTRONIC


Para acessar a placa SNMP do UPS e da linha GXT-MT da Vertiv, vamos configurar a placa de rede do computador com os seguintes dados:

  • IP: 192.168.102.220
  • MASCARA: 255.255.255.0

Agora vamos abrir o navegador e digitar o endereço padrão que é IP 192.168.102.230

Lembre-se de ultilizar a porta correta, indicada ao lado para conectar o cabo de rede.

Caso você não tenha sucesso na conexão com a placa,
vamos fazer reset da mesma, seguindo os passos abaixo:

1 - Remova a placa do slot;

2 - Altere o jumper jp1 para 2 e 3, conforme imagem abaixo:

3 - Inserir a placa no slot e aguardar de 3 a 5 minutos;

4 - Remover a placa do slot:;

5 - Retorna o jumper JP1 para 1 e 2;

6 - Inserir a placa no slot e aguardar 3 minutos;

7 - Configurar a placa conforme procedimento;

Ao acessar a placa, está será a primeira tela que veremos:


Descrisão:

A - Versão do SNMP Web Server.

B - Menu de funções. Oferece um conjunto de ferramentas para navegação e configuração da placa.

C - Tela principal. Ela exibirá todas as informações e/ou alternativas de controle de acordo com o menu selecionado.

D - Seção Login mostra o tipo de usuário de login atual. O usuário paddrão é


  • Senha: 12345678


Começando as configurações, vamos navegar até Sytem configuration -> Snmp configuration:



Em SNMP information, vamos preencher os campos:

  • SNMP equipament attached -> UPS 1
  • Contact -> UNION SISTEMAS E ENERGIA
  • Location -> Nome da Uniorg
  • System Name -> Número da Uniorg

Sempre em letra maiúscula



Agora vamos configurar o novo IP da placa Ainda em:

System Configuration -> snmp Configuration.

Em Network settings, vamos preencher os campos:

  • IP SETTINGS - Use a static IP addres
  • IP: 192.168.0.10
  • Mascara: 255.255.255.0
  • Gateway: 192.168.0.1

Clicar em APPLY.

Caso seja solicitado senha: 12345678



No campo SNMP CONFIGURATION -> TRAP IP ADDRESS

Vamos configurar:




TRAP IP Address é o responsável por enviar as informações do UPS via TRAP. Aqui, vamos configurar:

Trap IP Address 01: 192.168.0.50

Clicar em Apply

Ao final vamos reiniciar a placa clicando em Reboot The System.

Recomendo que acessem a placa novamente e verifiquem as configurações

*OBS - Para acessar a placa novamente, é necessário alterar o IP do seu computador dentro da nova classe de IP, por exemplo 192.168.0.29
